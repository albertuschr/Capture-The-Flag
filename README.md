# Web Exploit - CTF Writeups

This repository contains my personal writeups and notes for Web Exploitation challenges from Hack The Box and other CTF platforms. The goal of this repository is to document techniques, payloads, methodologies, and lessons learned during web security practice.

## Topics Covered

- SQL Injection (SQLi)
- Cross-Site Scripting (XSS)
- XML External Entity (XXE)
- Local File Inclusion (LFI)
- Remote File Inclusion (RFI)
- Server-Side Template Injection (SSTI)
- Command Injection
- Authentication Bypass
- File Upload Vulnerabilities
- Insecure Deserialization
- CSRF
- SSRF
- JWT Exploitation
- Directory Traversal
- API Exploitation
